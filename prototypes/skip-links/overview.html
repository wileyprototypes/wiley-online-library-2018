Skip Links

Accessibility standards recommended that adequate ‘skip links’ are provided to allow users that use keyboard navigation on WOL to quickly move to relevant sections on a page.

Resources

  • Video: 'Accessibility: Skip Links'
  • Article: '"Skip Navigation" Links'

Solution

Enhance the current skip link element to provide links to various points based on the page type.

Visually present the skip links to users when they have focus.

Notes

  • Some fo the pages on WOL are complex and include multiple layers of navigation and content - rather than implement a single skip link navigating the user to the 'main' content, we recommend creating a skip link menu that contains links to 2-3 most used areas of a particular page type

Designs and Specification

  • General Design (Journal Homepage)
  • To see skip links in action, press tab when the page has loaded
  • Recommend putting skip links in unordered list element to help imply there could be more than one
  • Skip links are visible when they have :focus
